# Music Library

Drop royalty-free background tracks here (.mp3 or .wav).

The editor picks a random track from this folder when mixing a reel.
Tracks should be:
- Royalty-free / licensed for social media use
- 60–180 seconds (longer tracks are trimmed automatically)
- Instrumental (no vocals — captions need to be audible)

Suggested sources:
- pixabay.com/music
- freemusicarchive.org
- soundcloud.com (Creative Commons)
